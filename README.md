# Personal Portfolio Website

A clean, modern personal portfolio website with smooth animations and responsive design.

## Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- Animated sections with intersection observer
- Clean and minimalist design
- Fast loading with vanilla JavaScript

## Structure

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - Interactive features and animations

## Customization

Replace the dummy content with your own:

1. **Personal Information**: Update name, title, and description in the hero section
2. **About Section**: Add your bio and update skills
3. **Experience**: Replace with your work history
4. **Projects**: Add your actual projects with links
5. **Contact**: Update email and social media links
6. **Colors**: Modify CSS variables in `styles.css` to match your brand

## Running Locally

Simply open `index.html` in your browser. No build process required!

## Deployment

You can deploy this site to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

Just upload the files and you're good to go!
